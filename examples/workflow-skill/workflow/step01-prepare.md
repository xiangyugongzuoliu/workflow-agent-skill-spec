# 步骤 01：准备

## 目标

解析目标目录，必要时创建运行目录，并记录审查配置。

## 输入

- 用户请求。
- 目标目录。

## 动作

1. 确认目标目录存在。
2. 列出顶层文件和文件夹。
3. 根据可见文件识别软件包类型。
4. 对需要检查多个文件的审查创建运行目录。
5. 写入 `state/config.json`。
6. 初始化 `state/progress.json`。

## 输出

```text
state/config.json
state/progress.json
step01-prepare/file-index.txt
```

## 完成标准

- 目标目录存在。
- 审查范围清晰。
- 进度状态中的 `current_step` 为 `step02-execute`。
- 下一步不需要重读完整会话即可运行。

## 失败处理

如果目标目录不存在，停止并要求用户提供有效路径。

如果目录过大，先只检查顶层结构；广泛扫描前先询问用户。
